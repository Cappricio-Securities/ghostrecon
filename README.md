## Installation

Install ghostrecon

```bash
  git clone https://github.com/karthi-the-hacker/ghostrecon.git
  cd ghostrecon
  npm install 
  pip install requests
  nano ~/includes/bin/github-subs.sh #replace with your github token
  mkdir ~/recon/ghostrecon/
```

## Usage

Start ghostrecon

```bash
  ./index.js
```

## Output

Check output in 

```bash
  ls ~/recon/ghostrecon/
```
